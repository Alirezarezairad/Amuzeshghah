## 🌈 درس 1: انواع داده در JavaScript

(Primitive & Reference Types)

---

### 🎯 هدف درس:

یاد بگیریم چطور جاوااسکریپت اطلاعات مختلف (مثل عدد، متن، لیست، آبجکت و تابع) رو درون خودش نگه می‌داره و تفاوت بینشون چیه.

---

## 🧩 بخش ۱ — انواع داده‌های اولیه (Primitive Data Types)

جاوااسکریپت مثل یک انبار بزرگه 🏠

داخلش جعبه‌های مختلفی هست که هرکدوم مخصوص نوع خاصی از داده‌ست.
به این‌ها می‌گیم **"انواع داده اولیه"** چون ساده و پایه‌ای هستن.

### 🔹 1. `string` — رشته (متن)

📘 یعنی نوشته‌هایی که داخل `" "` یا `' '` قرار می‌گیرن.

📌 مثال از کد:

```js
let name = "Hadi";
console.log(name);
console.log(typeof(name));
```

📖 توضیح ساده:

فرض کن یک دفتر داری برای نوشتن اسم‌ها ✍️

وقتی می‌نویسی `"Hadi"`، یعنی متنی داخل دفترت نوشتی.

🧠 `typeof(name)` می‌گه نوع داده‌ی متغیر چیه → در اینجا `"string"`.

✅ تمرین کوچک:

```js
let city = "Tehran";
console.log(typeof(city)); // خروجی: string
```

---

### 🔹 2. `number` — عدد

📘 یعنی داده‌هایی که برای محاسبات یا سن یا قیمت استفاده می‌شن.

📌 مثال:

```js
let age = 25;
console.log(age);
console.log(typeof(age));
```

📖 توضیح ساده:

عددها مثل دانه‌های پول هستن 💰

می‌تونی باهاش حساب و کتاب کنی.

⚠️ دقت کن: اگر عدد داخل "" باشه، دیگه عدد نیست، رشته‌ست!

```js
let notNumber = "25";
console.log(typeof(notNumber)); // string ❌
```

✅ تمرین:

```js
let price = 99.99;
console.log(typeof(price));
```

---

### 🔹 3. `boolean` — بولی (درست یا غلط)

📘 یعنی فقط دو حالت دارد: `true` (درست) یا `false` (غلط)

📌 مثال:

```js
let isCold = true;
console.log(typeof(isCold));
```

📖 توضیح ساده:

مثل چراغ برق 💡 که یا **روشن (true)** است یا **خاموش (false)**.

✅ تمرین:

```js
let isOnline = false;
console.log(typeof(isOnline));
```

---

### 🔹 4. `null` — عمداً خالی

📘 یعنی: الان چیزی نداره ولی قراره بعداً پر بشه.

📌 مثال:

```js
let person = null;
console.log(typeof(person)); // object (اشتباه تاریخی جاوااسکریپت)
```

📖 توضیح ساده:

مثل یک صندلی خالی 🪑 که فعلاً کسی روش ننشسته ولی بعداً می‌شینه.

✅ تمرین:

```js
let car = null;
console.log(car);
```

---

### 🔹 5. `undefined` — هنوز مقدار نگرفته

📘 یعنی متغیر ساخته شده ولی هنوز چیزی داخلش نیست.

📌 مثال:

```js
let job;
console.log(job);
console.log(typeof(job)); // undefined
```

📖 مثال از زندگی:

مثل جعبه‌ای که ساختی ولی هنوز چیزی توش نذاشتی 📦

✅ تمرین:

```js
let phone;
console.log(phone);
```

---

### 🔹 6. رشته‌ی خالی `""`

📘 یعنی متن داریم ولی عمداً خالی گذاشتیم.

📌 مثال:

```js
let persons = "";
console.log(typeof(persons)); // string
```

📖 مثال واقعی:
مثل دفترچه‌ای که عنوان داره ولی هنوز چیزی داخلش ننوشتی 📔

---

## 🌟 بخش ۲ — انواع داده‌های ارجاعی (Reference Data Types)

حالا می‌ریم سراغ داده‌هایی که خودشون ظرف بزرگ هستن و داخلش چند چیز دیگه دارن 🍱

---

### 🔹 1. `Array` — آرایه (لیست داده‌ها)

📘 یعنی مجموعه‌ای از چند داده پشت سر هم.

📌 مثال از کد:

```js
let number = ["4","OPEN-CODE",1,2,3,true,false,null,undefined];
console.log(number.length); // طول آرایه
console.log(number[1]); // دسترسی به عنصر دوم
```

📖 مثال واقعی:

مثل لیست خرید سوپرمارکت 🛒

هر مورد (نان، شیر، برنج) یک عنصر از آرایه است.

✅ تمرین:

```js
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[0]); // apple
```

---

### 🔹 2. `Object` — آبجکت (مجموعهٔ کلید و مقدار)

📘 یعنی هر ویژگی (کلید) همراه با مقدار خودش ذخیره می‌شود.

📌 مثال از کد:

```js
const personObject = {
  name: "Hadi",
  age: 25,
  job: "IT-Engineer",
  address: { city: "Tehran", country: "Iran" },
  hobbies: ["reading", "traveling"]
};
```

📖 مثال ساده:

مثل کارت شناسایی 🪪

اسم، سن، شهر و شغل — هرکدوم یک property (ویژگی) هست.

✅ تمرین:

```js
const car = {
  brand: "BMW",
  color: "black",
  year: 2025
};
console.log(car.brand);
```

---

### 🔹 افزودن و تغییر مقدار در آبجکت

📘 کاربرد: وقتی بخواهیم اطلاعات جدید اضافه یا آپدیت کنیم.

📌 مثال:

```js
personObject.email = "Hadi@gmail.com"; // افزودن
personObject.name = "Mehdi"; // تغییر مقدار
```

📖 مثال واقعی:

مثل وقتی شماره تلفن جدیدت رو به کارت ویزیت اضافه می‌کنی 📇

---

### 🔹 متدهای Object

📘 برای دسترسی به اطلاعات درون آبجکت‌ها:

```js
Object.keys(personObject);   // همه کلیدها
Object.values(personObject); // همه مقادیر
Object.entries(personObject); // جفت کلید-مقدار
```

📖 مثال واقعی:

مثل وقتی دفتر اسامی داری و هم می‌خوای فقط اسم‌ها رو ببینی، هم فقط شماره‌ها یا هر دو با هم 📚

---

### 🔹 3. `Function` — تابع

📘 یعنی بسته‌ای از کد که می‌تونه بارها استفاده بشه 🔁

📌 مثال:

```js
function myFunction() {
  console.log("Hello World!");
}
myFunction();
```

📖 مثال از زندگی:

مثل دستگاه قهوه‌ساز ☕

هر بار دکمه رو بزنی، کار خودش رو تکرار می‌کنه.

✅ تمرین:

```js
function sayHi() {
  console.log("Hi everyone!");
}
sayHi();
```

---

### تابع با ورودی

📘 می‌تونی اطلاعات مختلفی به تابع بدی.

```js
function myNameIs(name, age) {
  console.log("My name is " + name + " and I am " + age + " years old.");
}
myNameIs("Hadi", 25);
```

📖 مثال واقعی:

مثل فرم ثبت‌نام که اسم و سنت رو می‌گیری و چاپ می‌کنی 📝

---

### تابع با خروجی (`return`)

📘 یعنی نتیجه‌ای برگردونیم تا بعداً استفاده بشه.

```js
function square(number) {
  return number * number;
}
let nu = square(10);
console.log(nu);
```

📖 مثال واقعی:
مثل ماشین حساب 🧮 که وقتی عدد می‌دی، نتیجه رو برمی‌گردونه.

## 🧩 آزمون بخش (۱۵ سؤال)

1️⃣ نوع داده‌ی `"Hello"` چیست؟

2️⃣ خروجی `typeof(123)` چیست؟

3️⃣ مقدار `true` از چه نوع داده‌ای است؟

4️⃣ تفاوت `"25"` و `25` در چیست؟

5️⃣ مقدار `null` به چه معناست؟

6️⃣ خروجی `typeof(undefined)` چیست؟

7️⃣ آرایه چیست؟

8️⃣ چطور به عنصر سوم آرایه دسترسی پیدا می‌کنیم؟

9️⃣ آبجکت چیست؟

🔟 چطور یک property جدید به آبجکت اضافه می‌کنیم؟

1️⃣1️⃣ دستور `Object.keys(obj)` چه کاری انجام می‌دهد؟

1️⃣2️⃣ تابع چیست؟

1️⃣3️⃣ دستور `return` در تابع چه می‌کند؟

1️⃣4️⃣ اگر تابعی `console.log` درون خود داشته باشد ولی `return` نداشته باشد، چه برمی‌گرداند؟

1️⃣5️⃣ تفاوت `const` با `let` در تعریف آبجکت چیست؟
