# 🎓 درس ۶ — عملگرهای مقایسه‌ای در JavaScript

---

## 🎯 هدف درس

یاد بگیری چطور جاوااسکریپت با استفاده از «مقایسه‌ها» تصمیم می‌گیرد!
نتیجه همیشه یا **درست (true)** است یا **غلط (false)** 💭
مثل وقتی که می‌پرسی:

> آیا ۵ از ۱۰ کوچک‌تره؟ → بله ✅

> آیا ۵ از ۱۰ بزرگ‌تره؟ → نه ❌

---

## 🧩 بخش ۱ — تعریف ساده

📘 **عملگرهای مقایسه‌ای (Comparison Operators)** به ما کمک می‌کنن **دو مقدار** رو با هم مقایسه کنیم.

📌 خروجی همیشه از نوع **Boolean** هست یعنی فقط:

* ✅ `true` (درست)

* ❌ `false` (غلط)

📖 مثال از زندگی:
مثل ترازو ⚖️ که دو چیز رو با هم مقایسه می‌کنه تا ببینه کدوم سنگین‌تره یا مساویه!

---

## 🧩 بخش ۲ — انواع عملگرهای مقایسه‌ای

---

### 1️⃣ کوچکتر `<`

📘 بررسی می‌کنه آیا مقدار سمت چپ از سمت راست **کوچکتره**؟

📌 از کد تو:

```js
result = num1 < num2;   // 5 < 10 → true
console.log(result);
```

🎯 نتیجه: `true` ✅

📖 مثال:
مثل وقتی قد دو نفر رو مقایسه می‌کنی: «آیا قد علی از رضا کوتاه‌تره؟»

✅ تمرین:

```js
console.log(3 < 5); // true
console.log(10 < 2); // false
```

---

### 2️⃣ بزرگتر `>`

📘 بررسی می‌کنه آیا مقدار سمت چپ از سمت راست **بزرگتره**؟

📌 از کد تو:

```js
result = num1 > num2;   // 5 > 10 → false
console.log(result);
```

🎯 نتیجه: `false` ❌

📖 مثال:
مثل سؤال: «آیا ۵ سیب بیشتر از ۱۰ سیب است؟» 🍎

✅ تمرین:

```js
console.log(9 > 3); // true
console.log(3 > 9); // false
```

---

### 3️⃣ کوچکتر یا مساوی `<=`

📘 یعنی: آیا سمت چپ **کوچکتر یا مساوی** سمت راست است؟

📌 از کد تو:

```js
result = num1 <= num2;  // 5 <= 10 → true
```

🎯 نتیجه: `true` ✅

📖 مثال:
مثل «آیا سن تو از ۱۸ کمتر یا مساوی است؟» (برای ورود به باشگاه 😄)

✅ تمرین:

```js
console.log(10 <= 10); // true
console.log(8 <= 5);   // false
```

---

### 4️⃣ بزرگتر یا مساوی `>=`

📘 یعنی: آیا سمت چپ **بزرگتر یا مساوی** سمت راست است؟

📌 از کد تو:

```js
result = num1 >= num2;  // 5 >= 10 → false
```

🎯 نتیجه: `false` ❌

📖 مثال:

مثل اینکه بپرسی «آیا پولم از ۱۰۰ یورو بیشتر یا مساوی است؟» 💶

✅ تمرین:

```js
console.log(10 >= 10); // true
console.log(5 >= 7);   // false
```

---

### 5️⃣ مساوی `==`

📘 فقط **مقدار** رو مقایسه می‌کنه، نوع داده رو نه!

📌 از کد تو:

```js
result = num1 == num2;  // 5 == 10 → false
```

📖 مثال:

در جاوااسکریپت `5 == "5"` برابر است با `true` چون فقط مقدار را می‌بیند!

✅ تمرین:

```js
console.log(5 == "5"); // true
console.log(5 == 5);   // true
console.log(5 == 6);   // false
```

---

### 6️⃣ مساوی سختگیرانه `===`

📘 هم **مقدار** و هم **نوع داده** را مقایسه می‌کند.

📌 از کد تو:

```js
result = num1 === num2; // 5 === 10 → false
```

📖 مثال:

در این حالت `5 === "5"` ❌ چون یکی عدد است و یکی رشته (string).

✅ تمرین:

```js
console.log(5 === "5"); // false
console.log(5 === 5);   // true
```

---

### 7️⃣ نابرابر `!=`

📘 یعنی مقدارها **برابر نیستند** (نوع داده مهم نیست).

📌 از کد تو:

```js
result = num1 != num2; // 5 != 10 → true
```

🎯 نتیجه: `true` ✅

📖 مثال:

مثل اینکه بپرسی «آیا اسم من با تو فرق داره؟» 😄

✅ تمرین:

```js
console.log(10 != "10"); // false
console.log(5 != 6);     // true
```

---

### 8️⃣ نابرابر سختگیرانه `!==`

📘 یعنی مقدار یا نوع داده متفاوت است.

📌 از کد تو:

```js
result = num1 !== num2; // 5 !== 10 → true
```

📖 مثال:

در `5 !== "5"` → ✅ درست است چون یکی عدد است، یکی رشته.

✅ تمرین:

```js
console.log(5 !== 5);    // false
console.log(5 !== "5");  // true
```

---

## ✨ نکات آموزشی مهم از کدت

📌 `==` و `!=` فقط **مقدار** را چک می‌کنند (نوع را نه).

📌 همیشه بهتره از `===` و `!==` استفاده کنی تا مطمئن بشی هم مقدار و هم نوع داده یکی هست.

📌 خروجی همهٔ عملگرها **true یا false** است.

📌 این عملگرها بیشتر در **شرط‌ها (if, while, for)** استفاده می‌شن تا تصمیم بگیرن چه کاری انجام بشه.

## 🏠 تمرین‌های در خانه (۱۰ تمرین)

1️⃣ دو عدد انتخاب کن و با `<` و `>` بررسی کن کدام بزرگ‌تر است.

2️⃣ بررسی کن آیا سن تو از ۱۸ سال کمتر یا مساوی است یا نه (`<=`).

3️⃣ بررسی کن آیا نمره‌ات با عدد ۱۰۰ برابر است یا بیشتر (`>=`).

4️⃣ بررسی کن آیا `"5" == 5` و `"5" === 5` چه تفاوتی دارند.

5️⃣ خروجی `10 != "10"` و `10 !== "10"` را تست کن.

6️⃣ یک برنامه بنویس که بررسی کند آیا عدد واردشده **فرد** است یا **زوج**.

7️⃣ مقدار دو متغیر رشته‌ای (`"apple"` و `"banana"`) را با `==` و `===` مقایسه کن.

8️⃣ بررسی کن آیا متغیر `num1` کوچکتر یا مساوی `num2` است.

9️⃣ خروجی `3 > 3` و `3 >= 3` را چاپ کن و توضیح بده.

🔟 خودت سه مقایسهٔ مختلف بنویس که یکی `true` و دوتای دیگر `false` شوند.

## 🗒️ Cheat-Sheet (خلاصه تقلبی)

| عملگر | توضیح                    | مثال                | خروجی |
| ----- | ------------------------ | ------------------- | ----- |
| `<`   | کوچکتر                   | `5 < 10`            | true  |
| `>`   | بزرگتر                   | `5 > 10`            | false |
| `<=`  | کوچکتر یا مساوی          | `5 <= 10`           | true  |
| `>=`  | بزرگتر یا مساوی          | `5 >= 10`           | false |
| `==`  | مساوی فقط از نظر مقدار   | `5 == "5"` → true   |       |
| `===` | مساوی از نظر مقدار و نوع | `5 === "5"` → false |       |
| `!=`  | نابرابر (فقط مقدار)      | `5 != "5"` → false  |       |
| `!==` | نابرابر (مقدار یا نوع)   | `5 !== "5"` → true  |       |
