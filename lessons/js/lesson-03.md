# 🎓 درس ۳ — عملگرهای عددی و شیء Math در JavaScript

---

## 🎯 هدف درس:

یاد بگیری که چطور در جاوااسکریپت عددها رو جمع، تفریق، ضرب، تقسیم، یا حتی تصادفی تولید کنی 🎲

و بفهمی که شیء **Math** مثل یه ماشین حساب حرفه‌ای چطور بهت کمک می‌کنه 💪

---

## 🧩 بخش ۱ — عملگرهای عددی (Arithmetic Operators)

فرض کن داری یه مغازه کوچیک داری 🛍️

می‌خوای جمع قیمت‌ها، مانده حساب، یا مجموع امتیاز مشتری‌هارو حساب کنی!
اینجا جاوااسکریپت به کمکت میاد 😎

---

### ➕ جمع (`+`)

📘 برای اضافه کردن عددها به کار میره.

📌 مثال از کد:


```js
let num1 = 5;
let num2 = 10;
let result = num1 + num2;
console.log("جمع 5 و 10 => " + result);
```

🎯 خروجی: `15`

📖 مثال واقعی:

مثل وقتی ۵ تا سیب داری 🍎 و ۱۰ تا دیگه می‌خری، جمعاً ۱۵ تا سیب داری!

✅ تمرین:

```js
let apples = 3;
let bananas = 7;
console.log(apples + bananas);
```

---

### ➖ تفریق (`-`)

📘 برای محاسبه‌ی اختلاف دو عدد.

📌 مثال از کد:

```js
result = num1 - num2;
```

🎯 خروجی: `-5`

📖 مثال: اگه ۵ تا شیرینی داشتی و ۱۰ تا دادی رفت 🍩، دیگه ۵ تا بدهکار شدی 😅

✅ تمرین:

```js
let money = 100 - 35;
console.log(money);
```

---

### ✖️ ضرب (`*`)

📘 برای محاسباتی مثل مساحت، یا قیمت × تعداد.

📌 مثال از کد:

```js
result = num1 * num2; // 5 * 10 = 50
```

🎯 خروجی: `50`

📖 مثال: اگه هر بستنی 🍦 ۵ یورو باشه و ۱۰ تا بخری، می‌شی ۵۰ یورو بدهکار 😁

✅ تمرین:

```js
let price = 12;
let count = 4;
console.log(price * count);
```

---

### ➗ تقسیم (`/`)

📘 برای میانگین، نسبت‌ها یا تقسیم واحدها.

📌 مثال از کد:

```js
result = num1 / num2; // 5 / 10 = 0.5
```

📖 مثال: ۵ شکلات داری 🍫 و می‌خوای بین ۱۰ نفر تقسیمش کنی → هر نفر نیم شکلات می‌گیره!

✅ تمرین:

```js
let students = 10;
let candies = 50;
console.log(candies / students);
```

---

### 🔸 باقیمانده (`%`)

📘 برای فهمیدن باقی‌مانده‌ی تقسیم.

📌 مثال از کد:

```js
result = num1 % num2; // 5 % 10 = 5
```

📖 مثال: ۵ سیب رو بین ۱۰ نفر تقسیم کنی، به هر کسی چیزی نمی‌رسه ولی ۵ تا باقی می‌مونه 🍏

📌 کاربرد مهم: تشخیص عدد زوج و فرد

```js
let n = 7;
console.log(n % 2); // 1 = فرد
```

---

### ⚡ توان (`**`)

📘 یعنی عدد رو به توان عدد دیگه برسون.

📌 مثال از کد:

```js
result = num1 ** num2; // 5 به توان 10
```

📖 مثال: یعنی `5 × 5 × 5 × 5 ...` ده بار! 😮

✅ تمرین:

```js
let base = 2;
let power = 3;
console.log(base ** power); // 8
```

---

## 🧩 بخش ۲ — ترتیب انجام عملیات (Operator Precedence)

جاوااسکریپت یه قانون داره: بعضی عملگرها مهم‌ترن و زودتر انجام می‌شن 😎

📘 طبق کد:

* `**` (توان) از همه بالاتر است
* `*`، `/` و `%` بعدش میان
* `+` و `-` آخر
* پرانتز `()` همیشه اولویت بالاتر دارد 🎯

📌 مثال‌ها از کد:

```js
1 + 2 * 2   // 5
(1 + 2) * 2 // 6
1 + 2 ** 2  // 5
(1 + 2) ** 2 // 9
```

📖 مثال واقعی:

مثل آشپزیه 🍳؛ باید ترتیب کارها رعایت بشه—اول موادو آماده کن بعد بپز، نه برعکس!

✅ تمرین:

```js
console.log((2 + 3) * 2);
console.log(2 + 3 * 2);
```

---

## 🧩 بخش ۳ — خلاصه‌نویسی انتساب (Compound Assignment)

📘 برای کوتاه نوشتن تغییرات در مقدار متغیر:

```js
result += 1; // result = result + 1
result -= 1; // result = result - 1
result *= 2; // result = result * 2
result /= 2; // result = result / 2
result **= 2; // result = result ** 2
```

📖 مثال:

مثل وقتی داری حساب بانکی‌ت رو آپدیت می‌کنی 💰

به‌جای نوشتن جمله‌ی کامل، فقط می‌گی “اضافه کن!” 😄

✅ تمرین:

```js
let score = 10;
score += 5;
score *= 2;
console.log(score);
```

---

## 🧩 بخش ۴ — ++ و -- (افزایش یا کاهش یکی‌یکی)

📘 برای زیاد کردن یا کم کردن مقدار متغیر به اندازه‌ی ۱.

📌 مثال از کد:

```js
result++;
result--;
++result;
--result;
```

📖 مثال ساده:

مثل قدم زدن 🚶‍♂️

هر بار یه قدم می‌ری جلو (`++`) یا یه قدم برمی‌گردی عقب (`--`)

📌 تفاوت:

* `result++`: اول مقدار فعلی، بعد +۱

* `++result`: اول +۱، بعد مقدار جدید

✅ تمرین:

```js
let counter = 0;
counter++;
console.log(counter); // 1
```

## 🧩 بخش ۵ — شیء Math 🧮

📘 Math مثل یک جعبه ابزار جادویی پر از توابع و عددهای مهم ریاضی است ✨

---

### 🔹 ثابت‌های ریاضی:

```js
Math.PI  // عدد پی (۳٫۱۴)
Math.E   // عدد نپر (۲٫۷۱)
```

📖 مثال: وقتی می‌خوای محیط دایره حساب کنی، از `PI` استفاده می‌کنی 🍩

---

### 🔹 گرد کردن‌ها:

```js
Math.round(3.5); // 4 → نزدیک‌ترین
Math.ceil(3.5);  // 4 → همیشه بالا
Math.floor(3.5); // 3 → همیشه پایین
```

📖 مثال:

مثل وقتی قیمت ۳٫۵ یورو رو بخوای گرد کنی تا راحت‌تر حساب کنی 💶

---

### 🔹 قدر مطلق:

```js
Math.abs(-3.5); // 3.5
```

📖 مثال: یعنی "فقط اندازه" برام مهمه، نه علامتش!

---

### 🔹 توان و ریشه:

```js
Math.pow(3, 2);  // 3**2 = 9
Math.sqrt(9);    // √9 = 3
```

📖 مثال: یعنی «۳ به توان ۲» یا «ریشه دوم ۹» مثل درس ریاضی در مدرسه 🏫

---

### 🔹 کمینه و بیشینه:

```js
Math.min(0,1,2,...,10); // 0
Math.max(0,1,2,...,10); // 10
```

📖 مثال:
مثل پیدا کردن کوچک‌ترین و بزرگ‌ترین نمره در کلاس 📊

---

### 🔹 عدد تصادفی:

```js
Math.random(); // عددی بین 0 و 1
```

📖 مثال: مثل انداختن تاس 🎲

📌 الگوهای کد برای بازه‌های مختلف:

```js
Math.floor(Math.random() * 10);       // بین 0 تا 9
Math.floor(Math.random() * 10) + 1;   // بین 1 تا 10
Math.floor(Math.random() * (10 - 5 + 1)) + 5; // بین 5 تا 10
```

## 🏠 تمرین‌های در خانه (۱۰ تمرین)

1️⃣ برنامه‌ای بنویس که مجموع دو عدد را محاسبه کند.

2️⃣ برنامه‌ای بنویس که تفاضل دو عدد را نمایش دهد.

3️⃣ ضرب و تقسیم دو عدد مختلف را چاپ کن.

4️⃣ باقی‌مانده تقسیم سن خودت بر ۲ را چاپ کن (زوج یا فرد بودن سن).

5️⃣ عددی از کاربر بگیر و توان دوم آن را نمایش بده.

6️⃣ با استفاده از `+=` و `*=` مقدار یک متغیر را چند بار تغییر بده و در کنسول چاپ کن.

7️⃣ با استفاده از `++` یک شمارنده بساز که از ۰ تا ۵ بشمارد.

8️⃣ مقدار گرد شده، کف و سقف عدد ۷٫۳ رو چاپ کن.

9️⃣ کمترین و بیشترین عدد در مجموعه `[3, 9, 1, 5]` رو پیدا کن.

🔟 برنامه‌ای بساز که یک عدد تصادفی بین ۵ تا ۱۵ بسازد و در کنسول نمایش دهد.
